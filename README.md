# Staylog – 숙소 예약 플랫폼 (Frontend)
팀 프로젝트로 진행한 숙소예약 서비스 Staylog의 프론 저장소입니다. 
React + TypeScript를 기반으로 개발했으며, 반응형 UI 구성 및 REST API와의 연동을 통해 실제 예약 서비스에 가까운 동작을 구현했습니다.

# 주요기능
1) 메인페이지
- 지역별 숙소 목록 조회
- 정렬 옵션: 최신순 / 별점순 / 리뷰순 / 낮은가격순
- 숙소 카드 UI (이미지·평점·가격·리뷰 표시)
- 무한 스크롤(Infinite Scroll)로 숙소 목록 자동 로딩
2) 숙소/객실 상세 페이지
- 숙소 기본 정보 / 설명 표시
- 이미지 캐러셀(Image Carousel)
- Kakao Maps API 기반 지도 표시
3)날짜 선택 및 예약 기능
- Datepicker 기반 날짜 선택 UI
- 예약 불가일(disabled dates) 표시
- 인원 수 선택 (성인/어린이/영아)
4) 검색 필터 / 정렬
- 지역 코드 기반 숙소 필터링
5) 사용자 기능
- JWT 로그인 후 예약 가능
- 리뷰 작성/조회 (게시판과 연동)
6) 반응형 UI
- 모바일 / 태블릿 / 데스크톱 자동 대응
- Swiper / Carousel 기반 이미지 슬라이더 적용

# 역할
- 메인 페이지 섹션 구현
  -지역별 숙소 목록 API 연동
  - 정렬 옵션(최신/리뷰/별점/가격) 렌더링
  - 섹션별 숙소 카드 컴포넌트 구성
- 객실 상세 페이지 기능 개발
  - 객실 정보·이미지·가격·옵션 렌더링
  - 예약 불가일(disabled dates) 표시
- 예약 UX 구현
  - 체크인/체크아웃 UI
- Kakao Maps 연동(프론트 부분)
  - 주소 기반 좌표 표시
  - 숙소 상세 페이지에서 지도 렌더링
- 반응형 UI 개선
  - 모바일 레이아웃 조정
